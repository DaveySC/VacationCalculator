Проект был создан в рамках отбора в обучающий центр компании Neoflex.
Решение использует Java 11 + Spring Boot 2.7.4 + maven.

Данное приложение реализует один Rest Api по следующему пути : "/calculacte".
Контроллер этого Api принимает GET запросы со следующими параметрами:
	salary = int  (обязательный параметр)   (минимальное значение = 1)
	days   = int  (обязательный параметр)   (минимальное значение = 1)
	from   = date (необязательный параметр) (формат даты = dd-MM-yyyy)
	to     = date (необязательный параметр) (формат даты = dd-MM-yyyy)

Пример запроса: http://localhost:8080/calculacte?salary=30000&days=10&from=10-10-2022&to=20-10-2022

В результате вычислений возвращается сумма отпускных.

Некоторые пояснения :
	Вычисления отпускных происходит в заданном временном интервале не включая дату окончания отпуска.
	Дата from должна идти раньше даты to.
	Даты from и to не долнжы быть одинаковыми.
	При вводе полного списка параметров разница между датами from и to в днях должна быть равна параметру days.

Сборка проекта выполняется следующими командами:
	mvn clean package (компиляция проекта в jar файл)
	java -jar VacationCalculator-0.0.1-SNAPSHOT.jar (запуск Spring Boot приложения)
